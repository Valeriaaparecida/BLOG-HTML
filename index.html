Surgimento da internet

A internet surgiu durante a década de 1960, em meio ao contexto da Guerra Fria. O governo dos Estados Unidos financio u um projeto chamado ARPANET, desenvolvido pela ARPA, com o objetivo de criar uma rede de computadores capaz de continuar funcionando mesmo se parte da infraestrutura fosse destruída. Com o passar dos anos, essa rede começou a conectar universidades e centros de pesquisa, permitindo que cientistas compartilhassem informações de forma mais rápida. Na década de 1990, com a criação da World Wide Web por Tim Berners-Lee, a internet tornou-se mais acessível ao público, possibilitando a navegação por páginas e sites. Desde então, a internet evoluiu rapidamente e se tornou uma das tecnologias mais importantes do mundo, sendo utilizada para comunicação, estudo, trabalho e entretenimento. 🌐💻.

Como funciona o HTML

O HTML é a linguagem de marcação utilizada para estruturar o conteúdo das páginas da web. Ele funciona por meio de tags, que indicam ao navegador como cada parte do conteúdo deve ser organizada e exibida na tela. Quando um usuário acessa um site, o navegador interpreta o código HTML do documento e transforma essas marcações em elementos visuais, como títulos, parágrafos, listas, imagens e links. Dessa forma, o HTML não é responsável pela aparência avançada da página ou pela interatividade, mas sim pela estrutura e organização das informações. Cada elemento HTML é definido por uma tag de abertura e, na maioria dos casos, uma tag de fechamento. Entre essas tags fica o conteúdo que será exibido na página. Por exemplo, a tag p é usada para criar parágrafos, enquanto h1 até h6 são usadas para definir títulos de diferentes níveis. O HTML trabalha em conjunto com outras tecnologias da web, como CSS, responsável pelo estilo e aparência da página, e JavaScript, que adiciona interatividade e funcionalidades dinâmicas aos sites. Assim, o HTML é considerado a base de qualquer página da web, pois é ele que define a estrutura principal do conteúdo que será exibido no navegador. 🌐💻

O que são Tags

Tag <h1> até <h6> — Títulos e subtítulos

As tags <h1> até <h6> são utilizadas para definir títulos e subtítulos dentro de uma página HTML. Elas ajudam a organizar o conteúdo e tornar a leitura mais clara para o usuário.

A tag <h1> representa o título mais importante da página, normalmente o título principal do conteúdo. Já as tags <h2>, <h3>, <h4>, <h5> e <h6> são usadas para subtítulos em diferentes níveis de importância.

Essa estrutura é muito importante porque ajuda tanto os leitores quanto os mecanismos de busca, como o Google, a entenderem a organização da página.

Por exemplo, um artigo pode ter um <h1> como título principal, vários <h2> como seções e <h3> como subseções.

Exemplo:

<h1>Guia de Tags HTML</h1>
<h2>O que são tags HTML?</h2>
<h3>Exemplos de uso</h3>

Tag <p> — Parágrafo

A tag <p> é usada para criar parágrafos de texto em uma página HTML. Ela é uma das tags mais utilizadas, pois grande parte do conteúdo de um site é composta por texto.

Quando utilizamos a tag <p>, o navegador entende que aquele trecho é um parágrafo e automaticamente adiciona um pequeno espaço antes e depois do texto para melhorar a leitura.

Essa tag é muito importante para organizar textos em artigos, blogs, páginas institucionais e praticamente qualquer tipo de site.

Exemplo:

<p>HTML é a linguagem utilizada para estruturar páginas da web.</p>

Tag <mark> — Destacar texto

A tag <mark> é usada para destacar uma parte do texto, como se fosse um marca-texto utilizado em um caderno ou livro.

O navegador normalmente mostra o texto com um fundo amarelo, indicando que aquela parte tem algum destaque ou relevância dentro do conteúdo.

Essa tag pode ser utilizada para chamar atenção do leitor para palavras importantes ou informações que merecem destaque.

Exemplo:

<p>HTML é uma linguagem de <mark>marcação</mark> utilizada para criar páginas web.</p>

Tag <small> — Texto com tamanho menor

A tag <small> é usada para mostrar um texto em tamanho menor que o texto normal da página.

Ela costuma ser utilizada para observações, notas de rodapé, avisos ou informações complementares que não precisam ter tanto destaque quanto o conteúdo principal.

Apesar de o texto aparecer menor, ele continua sendo parte importante da página e pode ajudar a fornecer detalhes adicionais ao leitor.

Exemplo:

<p>Curso de HTML para iniciantes.</p>
<small>Conteúdo apenas para fins educacionais.</small>

Tag <i> — Texto em itálico

A tag <i> é utilizada para deixar o texto em itálico, ou seja, inclinado para a direita.

Esse tipo de formatação é comum quando queremos destacar palavras estrangeiras, títulos de obras ou expressões específicas dentro de um texto.

Exemplo:

<p>O termo <i>frontend</i> refere-se à parte visual de um site.</p>

Tag <u> — Texto sublinhado

A tag <u> serve para sublinhar um texto, colocando uma linha abaixo das palavras.

Ela pode ser usada para destacar algum trecho específico dentro de um parágrafo.

Exemplo:

<p>Este texto está <u>sublinhado</u>.</p>

Tag <strong> — Texto importante (negrito)

A tag <strong> é utilizada para indicar que um trecho do texto possui grande importância dentro do conteúdo.

Visualmente o navegador normalmente apresenta esse texto em negrito, ajudando a chamar a atenção do leitor.

Exemplo:

<p>Aprender HTML é <strong>muito importante</strong> para quem quer trabalhar com desenvolvimento web.</p>

Tag <ol> — Lista ordenada

A tag <ol> é utilizada para criar listas ordenadas, ou seja, listas em que os itens aparecem numerados.

Exemplo:

<ol>
<li>Abrir o editor de código</li>
<li>Criar um arquivo HTML</li>
<li>Escrever o código</li>
</ol>

Tag <ul> — Lista não ordenada

A tag <ul> cria listas com marcadores (bolinhas) e é usada quando a ordem dos itens não é importante.

Exemplo:

<ul>
<li>HTML</li>
<li>CSS</li>
<li>JavaScript</li>
</ul>

Tag <li> — Item da lista

A tag <li> representa um item dentro de uma lista e sempre aparece dentro de <ul> ou <ol>.

Exemplo:

<ul>
<li>Estudar HTML</li>
<li>Praticar exercícios</li>
<li>Criar projetos</li>
</ul>

Tag <a> — Links

A tag <a> é usada para criar links que permitem navegar entre páginas da internet ou partes de um site.

Exemplo:

<a href="https://www.google.com">Acessar o Google</a>

Tag <hr> — Linha horizontal

A tag <hr> cria uma linha horizontal usada para separar visualmente diferentes seções de conteúdo.

Exemplo:

<p>Primeira parte do texto</p>
<hr>
<p>Segunda parte do texto</p>

Tag <sub> — Texto subscrito

A tag <sub> mostra um texto um pouco abaixo da linha normal, muito usado em fórmulas químicas.

Exemplo:

<p>A fórmula da água é H<sub>2</sub>O.</p>

Tag <sup> — Texto sobrescrito

A tag <sup> coloca o texto um pouco acima da linha normal, sendo muito usada em potências matemáticas.

Exemplo:

<p>2<sup>3</sup> significa dois elevado à terceira potência.</p>

Tag <blockquote> — Citação

A tag <blockquote> é usada para representar citações mais longas de outros autores ou fontes.

Exemplo:

<blockquote>
A tecnologia move o mundo.
</blockquote>

Tag <font> — Definir fonte do texto

A tag <font> era utilizada em versões antigas do HTML para definir características da fonte de um texto, como cor, tamanho e tipo de letra.

Com ela era possível alterar a aparência do texto diretamente no HTML, definindo atributos como color, size e face.

No entanto, atualmente essa tag é considerada obsoleta e seu uso não é mais recomendado. Hoje em dia a formatação de texto deve ser feita utilizando CSS, que é a tecnologia responsável pelo estilo das páginas web.

Exemplo:

<font color="blue" size="4">Este texto está azul</font>

Tag <del> — Texto removido

A tag <del> é utilizada para indicar que um trecho de texto foi removido ou excluído de um conteúdo.

O navegador normalmente exibe esse texto com uma linha passando no meio das palavras, mostrando que aquela informação não é mais válida ou foi substituída.

Essa tag é muito utilizada em documentos, revisões de conteúdo ou comparações de versões de texto.

Exemplo:

<p>O preço antigo era <del>R$100</del> e agora é R$80.</p>

Tag <p> — Parágrafo

A tag <p> é utilizada para criar parágrafos de texto dentro de uma página HTML. Ela ajuda a organizar o conteúdo em blocos de leitura, tornando o texto mais claro e estruturado.

Quando um parágrafo é criado, o navegador automaticamente adiciona um pequeno espaço antes e depois do texto, melhorando a organização visual da página.

Essa tag é uma das mais utilizadas no desenvolvimento web, pois grande parte das informações em páginas e blogs é apresentada em forma de texto.

Exemplo:

<p>HTML é a linguagem utilizada para estruturar páginas da web.</p>

Tag <abbr> — Abreviação

A tag <abbr> é utilizada para representar uma abreviação ou sigla dentro de um texto.

Ela permite informar ao navegador e ao usuário qual é o significado completo daquela sigla, geralmente utilizando o atributo title.

Quando o usuário passa o mouse sobre a abreviação, o navegador normalmente mostra uma pequena explicação com o significado completo da palavra.

Essa tag é muito útil em textos técnicos, artigos e conteúdos educacionais.

Exemplo:

<p>Estou estudando <abbr title="HyperText Markup Language">HTML</abbr>.</p>
