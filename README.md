# 📚 Blog Educacional sobre HTML

Este projeto consiste em uma página web desenvolvida em **HTML** com o objetivo de explicar e demonstrar o funcionamento de diversas **tags fundamentais da linguagem HTML**.

O projeto foi criado como parte de um **desafio prático da DIO (Digital Innovation One)**, com o intuito de praticar a estruturação de páginas web e reforçar os conceitos básicos de HTML.

---

# 🌐 Sobre o Projeto

O blog apresenta explicações simples e exemplos práticos de várias tags HTML utilizadas na construção de páginas web.

Cada seção do blog contém:

- Explicação da tag
- Exemplo de uso
- Estrutura organizada para facilitar o aprendizado de iniciantes

O objetivo do projeto é ajudar estudantes iniciantes a compreenderem **como as tags HTML funcionam e como elas estruturam o conteúdo de uma página web.**

---

# 🧠 Tecnologias Utilizadas

- **HTML5**

Este projeto foi desenvolvido utilizando apenas **HTML puro**, sem o uso de CSS ou JavaScript, para focar no aprendizado da estrutura da linguagem.

---

# 🏷️ Tags HTML abordadas no projeto

O blog apresenta explicações e exemplos das seguintes tags:

- `<h1>` até `<h6>` — Títulos e subtítulos  
- `<p>` — Parágrafos  
- `<mark>` — Texto destacado  
- `<small>` — Texto em tamanho menor  
- `<i>` — Texto em itálico  
- `<u>` — Texto sublinhado  
- `<strong>` — Texto em negrito com importância semântica  
- `<ol>` — Lista ordenada  
- `<ul>` — Lista não ordenada  
- `<li>` — Item de lista  
- `<a>` — Links  
- `<hr>` — Linha horizontal  
- `<sub>` — Texto subscrito  
- `<sup>` — Texto sobrescrito  
- `<blockquote>` — Citações  
- `<font>` — Alteração de fonte (tag obsoleta no HTML moderno)  
- `<del>` — Texto removido  
- `<abbr>` — Abreviações

---

# 🎯 Objetivo do Projeto

O objetivo deste projeto é:

- Praticar a escrita de **HTML semântico**
- Aprender a **estruturar conteúdo web**
- Compreender o funcionamento das **principais tags HTML**
- Criar um pequeno **material educativo em formato de blog**
